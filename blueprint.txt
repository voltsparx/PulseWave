----------
Python  : orchestration, UI logic, search, settings

-----
✔ Minimalist retro CLI that feels like GUI
✔ Monochrome default, optional color themes
✔ Fast audio playback & seeking
✔ Beautiful ASCII audio visualizer
✔ Local + YouTube Music playback
✔ Keyboard-driven navigation
✔ Modular & future-proof
✔ Can compile into single binary later

========================
CORE MODULE RESPONSIBILITIES
========================

cli.py
------
- parses args
- loads config
- starts app loop

app.py
------
- main event loop
- handles input, render cycle, playback updates

player.py
---------
- play, pause, stop
- seek, rewind
- volume control
- communicates with C++ audio engine

queue.py
--------
- playlist management
- next/prev logic
- shuffle, repeat

search.py
---------
- local file search
- YouTube Music search
- fuzzy matching

config.py
---------
- theme mode
- color toggle
- keybindings
- audio buffer size

state.py
--------
- currently playing track
- playback time
- visualizer data



========================
C++ AUDIO ENGINE (native/audio)
========================

Responsibilities:
- low-latency playback
- decoding (mp3, flac, wav)
- buffering
- seek & rewind
- expose waveform data

Recommended libraries:
- PortAudio (cross-platform output)
- libsndfile (decoding)
- mpg123 (MP3 decoding)
- FFmpeg (optional advanced support)

Key Functions:
--------------
init_audio()
load_file(path)
play()
pause()
seek(seconds)
get_position()
get_waveform_chunk()



========================
DSP & VISUALIZER (native/dsp)
========================

Responsibilities:
- FFT processing
- amplitude extraction
- bar height generation

Pipeline:
---------
Audio → FFT → frequency bins → normalize → ASCII bar heights

ASCII levels:
-------------
▁▂▃▄▅▆▇█

Output:
-------
array<int> bar_heights



========================
CYTHON BRIDGE (native/bindings)
========================

pulsewave-11.pyx exposes:

- init_audio()

- seek(seconds)
- get_position()
- get_visualizer_data()

Purpose:
--------
Fast bridge between Python UI and C++ engine.
